# Claude_EDU — AI 프롬프트 허브

클로드 코드 공개과정 레포지토리입니다.

**라이브 데모**: https://LosTD-SW.github.io/Claude_EDU/

---

## 소개

AI 프롬프트 허브는 업무에 바로 적용할 수 있는 AI 프롬프트를 모아놓은 교육용 웹 애플리케이션입니다. 6개의 학습 모듈과 카테고리·난이도별 필터를 통해 원하는 프롬프트를 빠르게 찾고 활용할 수 있습니다.

## 주요 기능

- **6개 학습 모듈**: 프롬프트 기초, 비즈니스 커뮤니케이션, 데이터 분석, 마케팅 전략, 조직 관리, 전략 기획
- **카테고리 필터**: 마케팅, 데이터 분석, 고객 서비스, 문서 작성, 전략 기획, HR/채용
- **난이도 필터**: 초급 / 중급 / 고급
- **키워드 검색**: 프롬프트 제목·설명 실시간 검색
- **프롬프트 상세 보기**: 모달로 전체 프롬프트 및 예시 결과 확인

## 파일 구성

```
├── index.html        # 메인 웹 애플리케이션
└── prompt-data.js    # 프롬프트 데이터 및 모듈 설정
```

## 로컬 실행

별도 설치 없이 `index.html`을 브라우저에서 바로 열어 사용할 수 있습니다.
